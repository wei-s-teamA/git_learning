删除不要的分支:git branch -d|-D [分支名]
同步删除远端分支:git push origin -d [分支名]
新建分支:git checkout -b [分支名]
同步新增分支到远端:git push --set-upstream origin [分支名]
重命名文件:git mv [原名字] [新名字]
查看本地所有分支:git branch
查看本地所有分支，显示分支最近的commit信息:git branch -v
查看本地和远端所有分支:git branch -a
查看提交历史:git log --oneline --graph --all
打开图形化界面:gitk --all
修改最近提交的commit的message:git commit --amend
修改老旧commit的message:git rebase -i [所要修改的commit的父级commit的id]
比较工作区和暂存区:git diff
比较暂存区和本地最新一次commit的内容: git diff --cached
